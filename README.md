# homebrew-tap

Unofficial Homebrew tap for Eslzzyl's projects.

## Packages

| Package | Install |
|---------|---------|
| tidev | `brew install tidev` |
| Pixiv-SwiftUI | `brew install --cask pixiv-swiftui` |

## Usage

```bash
brew tap eslzzyl/tap
```

Then install any package listed above.

## Uninstall

```bash
brew untap eslzzyl/tap
```
